Run AI from your terminal

A fast, minimal CLI for running AI tasks from the command line or CI/CD pipelines. Written in Rust.

$ cargo install rai
terminal
# Ask anything directly
$ rai "Explain quantum computing in one sentence"
 
# Run a structured task from a Markdown file
$ rai task.md src/main.rs
 
# Run a specific sub-task
$ rai task.md "#security" src/main.rs
 
# Preview a task before running it
$ rai plan task.md
 
# Create a new task interactively
$ rai create review.md
Features

Markdown-driven tasks

Define reusable AI tasks in .md files with YAML frontmatter, templates, and sub-tasks.

Built-in tools

13 tools out of the box — shell, file I/O, HTTP, web search, git, and more.

CI/CD ready

Auto-detects non-interactive environments. Run AI tasks in your pipelines with -y flag.

Multi-model

Override the model per task or globally. Use GPT-4o, Kimi, or any supported provider.

Fast & lightweight

Written in Rust. Single binary, no runtime dependencies. Starts instantly.

Profiles & config

Multiple profiles for different workflows. TOML config with secure API key storage.

Quick start

1 Install 
cargo build --release
cp target/release/rai /usr/local/bin/
2 Set up your provider 
rai start
3 Run a task 
rai "Summarize this repo's README"
4 Create a reusable task file 
rai create review.md
rai review.md src/main.rs
